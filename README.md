# code-test-eduka
> Code test for eduka system internship.

## Install
- Python 3

## Run Local
```py
python nox.py
```
***Note*** : x is 1 or 2 or 3.

## Run online
[Online GDB](https://www.onlinegdb.com/online_python_compiler)

## Sample Problem

### Problem 1
- Input
```
12324
```
- Output
```
NO
```

### Problem 2
- Input
```
121 112
```
- Output
```
YES
```

### Problem 3
- Input
```
abbacda
```
- Output
```
0 3
```
